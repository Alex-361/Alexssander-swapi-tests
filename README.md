# Projeto SWAPI Testes

Este projeto foi desenvolvido para implementar e automatizar testes de API utilizando a Star Wars API (SWAPI). Utilizamos Node.js juntamente com as bibliotecas Axios para requisições HTTP e Jest para a criação e execução dos testes. O objetivo é garantir a integridade e funcionalidade dos endpoints da SWAPI, bem como verificar comportamentos para rotas inexistentes.

Além disso, configuramos workflows no GitHub Actions para executar automaticamente cada teste sempre que houver um push ou pull request no repositório. Isso assegura que quaisquer alterações no código não quebrem a funcionalidade existente e mantêm a qualidade do projeto de forma contínua.
